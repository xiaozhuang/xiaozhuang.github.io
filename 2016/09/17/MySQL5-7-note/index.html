MySQL5.7安装及新特性

安装

windows版

系统：win7x64安装
MySQL：5.7.15

解压mysql-5.7.15-winx64.zip

复制my-default.ini命名为my.ini，创建保存数据data文件夹

管理员运行cmd，进入mysql下bin目录，初试化数据

安装MySQL为windows服务

启动MySQL

将mysql添加到系统环境变量

添加path环境变量添加：

添加系统环境变量MYSQL_HOME，指向mysql根目录

注：
与Linux版MySQL5.7不同，windows版MySQL5.7不会初试化随机密码
windows版MySQL5.7安装完成后，密码为空

未开启同步时状态
Slave_IO_Running: No
Slave_SQL_Running: No

同步状态信息报错
Last_IO_Error: Fatal error: The slave I/O thread stops because master and slave have equal MySQL server UUIDs; these UUIDs must be different for replication to work.
因虚拟机拷贝mysql/data/auto.cnf中server-uuid一样

同步状态中Slave_IO_Running，Slave_SQL_Running同时为yes，表示同步正常
