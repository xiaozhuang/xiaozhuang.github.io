路由器当作交换机用

前端路由器—-LAN口—-路由器—-LAN口—-计算机

  • 打开路由器的设置界面，把无线路由器的DHCP 服务关掉

端口映射

任何到本地端口的TCP连接都能够被转发到任意一个本地端口，或远程主机的某个端口

添加一个IPV4到IPV4的端口映射
netsh interface portproxy add v4tov4 listenaddress=127.0.0.1 listenport=3306 connectaddress=192.168.101.53 connectport=3306

指定监听ip和端口可以删除
netsh interface portproxy delete v4tov4 listenaddress=127.0.0.1 listenport=3306

查看转发了哪些端口
netsh interface portproxy show all

添加路由

打印所有route
route print

route add 192.168.101.53 MASK 255.255.255.255 127.0.0.1
route delete 192.168.101.53
