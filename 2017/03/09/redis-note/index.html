Redis Note

发表于 2017-03-09   |   分类于 Note

Redis是一个远程内存数据库，不仅性能强劲，还具有复制特性以及为解决为题而生的独一无二的数据模型。

Redis

Redis是非常高性能和轻量级的数据存储。
提供了键/值的数据访问方式来持久化数组、列表、集合以及哈希数据结构部。
支持原子级的计数器(counter)并具有高效的基于主题的发布/订阅消息功能。

Geode与Redis

Apache Geode是一个相当成熟、强健的的数据管理平台，提供实时的、一致的、贯穿整个云架构地访问数据关键型应用。

Geode自身功能比较多，首先它是一个基于JVM的NoSQL分布式数据处理平台，同时集中间件、缓存、消息队列、事件处理引擎、NoSQL数据库于一身的分布式内存数据处理平台。可用来进行完成分布式缓存、数据持久化、，分布式事物、动态扩展等功能。

Redis 数据结构使用场景

Redis命令

Redis中的字符串(string)

命令 作用
get 获取
set 设置
del 删除（可用于所有类型）
setnx 将 key 的值设为 value，当且仅当 key 不存在

自增和自减命令

如果对不存在的键或者保存了空串的键执行自增或者自减操作，Redis将值当做0处理

命令 作用
incr 值加一
decr 值减一
incrby 加上整数
decrby 减去整数
incrbyfloat 加上浮点数

处理子串和二进制位的命令

命令 作用
append 追加到末尾
getrange 获取指定范围子串
setrange 从指定位置开始子串设定为给定值
getbit 字符串看做二进制位串，返回指定偏移量的二进制值
setbit 设置二进制位的值
bitcount 统计二进制位串里值为1的二进制数量
bitop 对一个或多个二进制串执行并（AND）、或（OR）、异或（XOR）、非（NOT）按位操作

过期时间

EXPIRE key seconds 为给定 key 设置生存时间，当 key 过期时(生存时间为 0 )，它会被自动删除。

Redis中列表(list)

Redis对链表（linked-list）结构的支持使得它在键值存储的世界中独树一帜

命令 作用
rpush 推入左端
lpush 推入右端
lpop 左端弹出
rpop 右端弹出
lindex 获取列表指定位置元素
lrange 获取给定范围内元素（包含偏移量两端元素,0,-1返回全部）
ltrim 修剪列表（偏移量两端的元素会保留）

阻塞是列表弹出命令以及在列表之间移动元素的命令

阻塞弹出命令和弹出并推入命令，常见用例是消息传递（messageing）和任务队列（task queue）

命令 作用
blpop 弹出左端元素 或阻塞等待
brpop 弹出右端元素 或阻塞等待
rpoplpush 从源列表右侧弹出，推入目标列表左侧，并返回该元素
brpoplpush 从源列表右侧弹出，推入目标列表左侧，并返回该元素 或阻塞等待

Redis集合(set)

命令 作用
sadd 添加元素
srem 移除元素
scard 元素数量
sismember 检查是否存在
srandmember 随机返回一个或多个元素
smembers 获取所有（注：若元素非常多，会很慢，谨慎使用）
spop 随机移除一个元素，并返回该元素
smove 从源集合移动至目标集合

用于组合和处理多个集合的Redis命令

命令 作用
sdiff 返回存在于第一个集合，不存在与其他集合的元素（差集计算）
sdiffstore 差集计算并存储结果
sinter 同时存在于所有集合中的元素（交集计算）
sinterstore 交集计算并存储结果
sunion 至少存在于一个集合中的元素（并集计算）
sunionstore 并集计算并存储结果

Redis的散列(hash)

命令 作用
hset 添加一个键值对
hmset 设置一个或多个键的值
hget 获取单个键的值
hmget 获取一个或多个键的值
hdel 移除键值对
hgetall 获取所有键值对
hlen 散列包含价值对数量

散列批处理命令
命令 | 作用
————|————–
hexists | 检查指定键是否存在于散列中
hkeys | 获取散列所有键
hvals | 获取散列所有值
hgetall | 散列包含的所有键值对
hincrby | 将键保存的值加上整数
hincrbyfloat | 将键保存的值加上浮点数

Redis的有序集合(zset)

存储键值对，键被称为成员(member)，值被称为分值(score),分值必须为浮点数
既可以根据成员访问元素，又可以根据分值及分值排序访问元素

命令 作用
zadd 添加成员
zrem 移除成员
zcard 成员数量
zincrby 增加成员分值
zrange 根据排序位置，获取集合中元素
zrangebyscore 获取给定分值范围内元素
zcount 分值介于min和max之间的成员数量
zsore 返回成员分值
zrank 获取成员排名

共享锁、排他锁

X锁 eXclusive
S锁 Share
